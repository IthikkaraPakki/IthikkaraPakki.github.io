High frequency sampling - 7.5μS

Host-Target model approach in SIMULINK reduces load on the target hardware and enables faster signal sampling rates

STM32 SIMULINK Code generation Embedded Coder DSP
  • Native Simulink external mode for real-time signal sampling introduces more load on the target hardware
  • A two model approach one for target and one for host reduces the load enabling high signal sampling rates
  • This can give sampling rates upto 7.5μS on 7.5Mbps USART for STM32 Microcontrollers
  • This project developed custom Simulink model for STM32G431 and streamed 2Mbps via USART to host

Field Oriented Control — FreeRTOS + ISR

Speed control of BLDC motor using FOC for increased efficiency and performance. Implemented on STM32 with real-time state machine management on FreeRTOS and safety critical motor control logic inside high-priority ISRs.

STM32 Embedded C FreeRTOS AS5600 PID FOC I2C USART
  • Implemented inverse Clarke & Park transforms for voltage vector control.
  • Startup sequence runs "encoder offset calibration" and "eccentricity calibration"
  • Speed control loop(PI) runs at 1KHz and PWM at 20KHz. Motor control and safety logic runs inside ISR.
  • State-machine update using FreeRTOS and rest of the application using LL_drivers.
  • Segger SystemView for verifying execution time and ISR safety.

Low-Side Current Sensing — 3-Phase Inverter

Current sensing is required for safety in control circuits especially inverters and for torque control in BLDC motor drive applications. Implemented on B-G431-ESC1 STM32G431 evaluation board.

STM32 B-G431-ESC1 BLDC ADC USART DSP
  • ADC-based low-side shunt resistor current measurement on all three phases
  • Current is measured exactly when low side switches are On
  • ADC PWM synchronization using TRGO
